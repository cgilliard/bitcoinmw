Expand description

A Lock which can be used to send data to and from threads which is also stored in a box so it can be stored in structures as well. See crate::lock_box.

fn danger_to_usize(&self) -> usize

consume the inner Arc and return a usize value. This function is dangerous because it potentially leaks memory. The usize must be rebuilt into a lockbox that can then be dropped via the crate::lock_box_from_usize function.
